Base URL for Open Traffic data files:

http://ph-opentraffic-data.s3-website-ap-southeast-1.amazonaws.com/

Data is divided into a /cebu and /manila directory. Each city has:
 - a Shapefile export containing geomety and edge_id for each road segment
 - a set of weekly CSV data files (date in name is first day of week)

Shapefile and CSV data can be linked using the edge_id column in the CSV. 

Join filenames below with base URL (e.g. http://ph-opentraffic-data.s3-website-ap-southeast-1.amazonaws.com/cebu/week_2015-12-28.csv.gz)
